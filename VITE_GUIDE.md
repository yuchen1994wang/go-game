# Vite 开发指南

## 安装

```bash
npm install
```

## 开发

### 启动开发服务器
```bash
npm run dev
```

启动后访问 http://localhost:3000

### 启动指定页面
```bash
npm run dev:game  # 直接打开 game.html
```

## 构建生产版本

```bash
npm run build
```

构建产物会输出到 `dist` 目录。

## 预览生产构建

```bash
npm run preview
```

## Vite 特性

- ⚡ 极快的开发服务器启动
- 🔥 真正的热模块替换 (HMR)
- 📦 高效的生产构建
- 🎯 支持多页面应用
- 🛠️ 开箱即用的 TypeScript 支持
