# 围棋对练复盘 微信小程序

双人同屏围棋对弈，支持每步点评和复盘回放。

## 功能特性

- **多种棋盘**: 9路/13路/19路可选
- **围棋规则引擎**: 自动提子、禁着点、打劫判定
- **玩家名称**: 记录黑方白方名称
- **每步点评**: 对每一步进行文字点评
- **复盘回放**: 逐步查看对局，支持跳转
- **棋谱导出**: SGF格式导出

## 项目结构

```
weapp/
├── app.js              # 应用入口
├── app.json            # 应用配置
├── app.wxss            # 全局样式
├── pages/
│   ├── index/          # 首页（选择棋盘大小）
│   └── game/          # 对弈页面
└── project.config.json # 项目配置
```

## 开发运行

1. 下载 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)

2. 打开微信开发者工具，点击「导入项目」

3. 选择 `weapp` 文件夹作为项目目录

4. 填写 AppID（可使用测试号）或选择「体验小程序」

5. 点击「确定」即可在模拟器中预览

## 提交审核发布

1. 在微信开发者工具中点击「上传」

2. 登录 [微信公众平台](https://mp.weixin.qq.com/)

3. 进入「开发管理」→「版本管理」

4. 提交审核并等待通过

## 棋谱导出

导出的SGF格式可在以下工具中打开：
- Katrain
- Lizzie
- SmartGo
- 其他支持SGF的围棋软件

## 技术说明

- 使用原生微信小程序框架开发
- Canvas绘制棋盘和棋子
- 支持触屏落子
